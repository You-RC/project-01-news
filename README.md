# Stock Market News App

A Next.js web application that fetches and displays the most important daily news affecting stock markets using free RSS feeds.

## Quick Start

- Run the included script: `./run.sh`
- Or in VS Code: Ctrl+Shift+P > Tasks: Run Task > Dev Server
- Or manually: `npm run dev`

Then open [http://localhost:3000](http://localhost:3000).

## Features

- Fetches daily business news from Reuters RSS feed
- Displays articles with title, summary, date, and links
- No sign-up or API keys needed
- Responsive design with Tailwind CSS
